
# ytdlp-to-show
Input URLs, output a media library ready for import, containing a faux-"TV Series" with Season (year) and Epsiode (1 onward, by date)

## Current status: Very WIP. Working, but in active development. No documentation/limited messaging/subject to breaking change.


Now:
- Unique ID added by website URL domain and video ID for solid watch state management (eg. Kodi, Jellyfin, ...)
- Title and Description
- Thumbnails/channel artwork (WIP)
- Subtitles (when provided upstream)

(Intended) Future:
- Reprocess prior-downloaded media by folder w/ intelligent title matching
- Re-titling
- ...
